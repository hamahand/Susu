# SusuSave Logo & Brand Assets

This folder contains all the logo and brand assets for the SusuSave web application.

## Logo Files

### Main Logo
- **logo.svg** - Full horizontal logo with icon and text (200x60px)
  - Use in: Navigation bar, marketing materials, documentation
  - Color: Green gradient (#2E7D32 to #047857) with dark text

### Logo Icon
- **logo-icon.svg** - Standalone circular logo icon (64x64px)
  - Use in: App icons, small spaces, social media avatars
  - Features: Three overlapping circles representing community + Cedi symbol (₵)

## Favicon Files

- **favicon.svg** - Main favicon (32x32px)
- **favicon-16x16.svg** - Small favicon for older browsers
- **favicon-32x32.svg** - Standard favicon size
- **apple-touch-icon.svg** - Apple device home screen icon (180x180px)

## Social Media Assets

- **og-image.svg** - Open Graph image for social media sharing (1200x630px)
  - Use in: Facebook, LinkedIn, Twitter card previews
  - Includes: Logo, tagline, and key statistics

## Brand Colors

### Primary Colors
- **Green Primary**: `#2E7D32` (Material Green 800)
- **Green Dark**: `#059669` (Emerald 600)
- **Green Darker**: `#047857` (Emerald 700)

### Usage
- Primary green represents growth, prosperity, and savings
- Three overlapping circles symbolize community and collective savings
- Cedi symbol (₵) represents the Ghanaian currency and local relevance

## Design Concept

The logo combines three key elements:
1. **Community**: Three overlapping circles representing group savings
2. **Currency**: The Cedi symbol (₵) at the center
3. **Growth**: Green gradient suggesting financial growth and prosperity

The design reflects SusuSave's mission: bringing Ghanaian communities together through modern savings technology.

## File Formats

All assets are provided as SVG (Scalable Vector Graphics) for:
- ✅ Perfect scaling at any size
- ✅ Smaller file sizes
- ✅ Better performance
- ✅ Easy editing and customization

## Converting to PNG

If you need PNG versions, you can:
1. Open the SVG in a browser
2. Use an online converter like [CloudConvert](https://cloudconvert.com/svg-to-png)
3. Use design tools like Figma, Illustrator, or Inkscape
4. Use command line: `convert logo.svg -resize 512x512 logo.png` (requires ImageMagick)

## Usage Guidelines

- ✅ DO: Use the logo with adequate spacing around it
- ✅ DO: Maintain the aspect ratio when resizing
- ✅ DO: Use on light backgrounds (white or light gray)
- ❌ DON'T: Distort or stretch the logo
- ❌ DON'T: Change the brand colors
- ❌ DON'T: Add effects like shadows or outlines
- ❌ DON'T: Use on busy or dark backgrounds without adjustment
