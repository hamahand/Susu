┌────────────────────────────────────────────────────────────────┐
│  STEP 3: ADD UTM TRACKING PARAMETERS                          │
└────────────────────────────────────────────────────────────────┘

Add these to your affiliate links for better tracking:

?utm_source=sususave&utm_medium=affiliate&utm_campaign=homepage

EXAMPLE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Basic link:
https://partner.com/?ref=YOUR_ID

Enhanced link:
https://partner.com/?ref=YOUR_ID&utm_source=sususave&utm_medium=affiliate&utm_campaign=homepage

Benefits:
✓ Track which page drove the conversion
✓ See which campaign performs best
✓ Analyze in Google Analytics


┌────────────────────────────────────────────────────────────────┐
│  STEP 4: VERIFY LINKS WORK                                    │
└────────────────────────────────────────────────────────────────┘

After replacing, test each link:

□ 1. Click the link
□ 2. Check it opens affiliate partner site
□ 3. Check URL contains your affiliate ID
□ 4. Verify tracking cookie is set
□ 5. Check affiliate dashboard shows the click


┌────────────────────────────────────────────────────────────────┐
│  AFFILIATE LINK BEST PRACTICES                                │
└────────────────────────────────────────────────────────────────┘

✓ Always use rel="nofollow sponsored"
✓ Always use target="_blank" (opens new tab)
✓ Add UTM parameters for tracking
✓ Use descriptive link text ("Apply Now" not "Click Here")
✓ Test links monthly (they can expire)
✓ Check conversion tracking works
✓ Monitor click-through rates
